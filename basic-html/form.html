สรุป form แต่ละชนิด


Tag form ก็จะมีหลาย element ให้เราได้เลือกใช้ เวลาจะใช้ค่อยมาเปิดดูเอา ซึ่งหลักๆแล้วตัวที่ได้ใช้บ่อยๆก็คงจะมีไม่กี่ตัว
Form แต่ละชนิดมีดังนี้
1.  <input> เป็น element ที่ใช้บ่อยๆ Link ตัวอย่างเพิ่มเติม
2.  <label> เป็นชื่อหน้าช่อง Form ที่เราใช้ระบุว่าช่องแบบฟอร์มที่ว่าเป็นช่องแบบฟอร์มอะไร
3.  <select> เป็น element form drop down list
4.  <textarea> เป็นฟอร์มที่เอาไว้ ส่งข้อความทั่วไป
5.  <button> ปุ่มกด
6.  <fieldset> ใช้คู่กับ legend
7.  <legend> ใช้คู่กับ feildset อ่านเพิ่มเติม การใช้ทั้งคู่
8.  <datalist> คล้ายๆกับ select เลย
9.  <output> อันนี้น่าสนใจน่าเอาไปประยุกต์ใช้กับเว็บไซต์แนวๆเลื่อนๆแถบแสดงจำนวน
